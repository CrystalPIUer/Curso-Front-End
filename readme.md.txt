Es una practica de cómo crear una página web. Tiene elementos de semántica básica y lo vamos a ir mejorando a medida que avancen las clases.

En este caso es un carrito de compra que ofrece artículos de crossfit, como ropa o implementos deportivos. Ofrecemos un catalogo, medios de pago y formas de envío, además de un formulario de contacto ante dudas.
